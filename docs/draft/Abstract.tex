\begin{enumerate}
\item ネットワーク科学が注目を集めて久しいのです
\item 用途に応じた中心性がいろいろあるのです

\item その中でも，
  \begin{itemize}
  \item Single Source Shortest Path (SSSP) DAGを保存する
  \item RamalingamとRepsのアルゴリズムに基づく

\item われわれはかしこいので
\end{enumerate}
